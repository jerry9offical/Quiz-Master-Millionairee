# Audio Assets

This folder contains royalty-free audio files for QuizMaster.

## Audio Files (Downloaded)

| File | Description | Source |
|------|-------------|--------|
| `bg_loop.mp3` | Quiz show background music loop | OrangeFreeSounds.com |
| `bg_ambiance.mp3` | Soft ambient background | (Uses bg_loop as fallback) |
| `sfx_correct.mp3` | "Ta-Da!" correct answer sound | SoundBible.com |
| `sfx_wrong.mp3` | Sad trombone wrong answer | SoundBible.com |
| `sfx_levelup.mp3` | Power-up level advance sound | SoundBible.com |
| `sfx_lock.mp3` | Glass ping for answer lock-in | SoundBible.com |
| `sfx_tick.mp3` | Clock tick for countdown | SoundBible.com |

## Usage Notes

- Background music loops seamlessly
- All audio is controlled via the AudioManager
- User preferences are stored in localStorage
